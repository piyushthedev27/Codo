# 🎮 CODO — Code Duel Online

[![Next.js](https://img.shields.io/badge/Next.js-14-black?logo=next.js)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue?logo=typescript)](https://www.typescriptlang.org/)
[![Firebase](https://img.shields.io/badge/Firebase-v11-orange?logo=firebase)](https://firebase.google.com/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC?logo=tailwind-css)](https://tailwindcss.com/)

> **A pixel-retro RPG-themed coding education platform where you never learn alone.**

CODO (Code Duel Online) is a gamified, competitive coding education platform that combines real-time coding challenges, structured learning, and social features into a single unified application. Built with **Next.js 14**, **Firebase**, and **Anime.js**, it offers a premium, immersive experience for developers to level up their skills.

---

## ✨ Key Features

### 🤖 AI-Powered Learning
- **AI Study Partners**: Learn alongside Sarah (Frontend), Alex (Algorithms), or Jordan (Systems) — each with distinct personalities and specialties.
- **AI Code Cinema**: Interactive, animated code explanations with playback and voiceover that make complex concepts easy to grasp.
- **Mistake Analyzer**: Post-submission analysis that categorizes error patterns (off-by-one, null pointers, etc.) and provides targeted improvement suggestions.

### 🎮 Gamified Experience
- **World Map Navigation**: Explore coding topics as pixel art adventure zones.
- **Quest System**: Complete daily and weekly challenges to earn rewards and stay consistent.
- **XP & Leveling**: Earn experience points from every activity and watch your rank climb from Novice to Legend.
- **Pixel Pet Companion**: A virtual companion that grows and evolves as you progress through your learning journey.

### ⚔️ Competition & Community
- **Code Duels**: Compete in timed, head-to-head coding challenges against AI peers or global players.
- **Guilds**: Join or create teams of 5 members to collaborate on weekly goals and dominate the guild leaderboards.
- **Knowledge Graph**: A visual skill tree that tracks your mastery across different technologies.

---

## 🛠️ Technology Stack

| Component | Technology | Purpose |
|-----------|------------|---------|
| **Core Framework** | Next.js 14 (App Router) | Unified React frontend + API routes |
| **Logic** | TypeScript | Strong typing and developer productivity |
| **Styling** | Tailwind CSS | Modern, responsive pixel-retro UI |
| **Animation** | Anime.js | Fluid micro-animations and RPG effects |
| **Database/Auth** | Firebase | Real-time Firestore & secure authentication |
| **Testing** | Jest + Fast-Check | Unit & property-based reliability testing |

---

## 🚀 Quick Start

Get the project running locally in under 5 minutes:

1.  **Clone and Install**:
    ```bash
    npm install
    ```
2.  **Environment Setup**:
    ```bash
    cp .env.example .env.local
    # Configure your Firebase keys in .env.local
    ```
3.  **Launch**:
    ```bash
    # Terminal 1: Firebase Emulators
    npm run emulator

    # Terminal 2: Development Server
    npm run dev
    ```

For detailed instructions, see the **[Quick Start Guide](docs/QUICK_START.md)**.
