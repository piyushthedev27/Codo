# Codo - AI-Powered Peer Learning Platform

Codo is an innovative platform designed to revolutionize programming education through synthetic AI companions. Instead of learning in isolation, users interact with personalized AI peers who have unique personalities, learning styles, and coding expertise.

## 🌟 Key Features

- **AI Peer System**: Interact with synthetic learners like Sarah, Alex, and Jordan, each designed to complement different learning styles.
- **Collaborative Coding**: Real-time shared editor for peer coding, duels, and interactive debugging.
- **Voice Coaching**: Browser-native speech recognition and synthesis for hands-free guidance.
- **Learning Dashboard**: A personalized command center to track your progress, achievements, and peer activity.
- **Interactive Knowledge Graph**: Visualize your learning path and understand the relationships between coding concepts.

## 🚀 Getting Started

To get the application running locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment**:
   Copy the example environment file and fill in your keys:
   ```bash
   cp .env.example .env.local
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to see the application in action.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Database**: [Supabase](https://supabase.com/)
- **Authentication**: [Clerk](https://clerk.com/)
- **AI**: [OpenAI API](https://openai.com/)
- **Styling**: Vanilla CSS & Tailwind CSS
- **Animations**: Framer Motion
